# MAC (Mandatory Access Control)

Mandatory Access Control (MAC) is a security model where access rights are regulated by a central authority based on multiple levels of security. This section covers various MAC models implemented in Casbin.

## Available Models

Explore the MAC models available in Casbin:

- [MAC Overview](/docs/mac-overview) - Introduction to Mandatory Access Control
- [BLP (Bell-LaPadula)](/docs/blp) - Confidentiality-focused model
- [Biba](/docs/biba) - Integrity-focused model
- [LBAC (Lattice-Based Access Control)](/docs/lbac) - Hierarchical security levels
